# References

This directory contains bibliography files and citation materials for presentations.

## Organization

- **general-bibliography.bib** - General references for mathematical music topics
- **by-topic/** - Topic-specific bibliography files
- **citation-styles/** - Citation style files for different formats

## Citation Management

Recommended tools:
- Zotero
- Mendeley
- EndNote
- BibTeX/BibLaTeX

## Style Guidelines

Use consistent citation formats:
- IEEE for technical papers
- APA for educational/psychological research
- Chicago for musicological work
- Custom MMS style when specified

## File Formats

- `.bib` files for BibTeX
- `.ris` files for reference manager import
- `.json` files for web-based bibliography tools
